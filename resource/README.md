# 參考資料 References

本目錄收集了本專案開發時所參考的所有紙本資料。
多數為網路上蒐集而來，新加坡3份簡體字表則為2019年赴新加坡國家圖書館借閱之館藏。

盡可能想蒐集到掃描的原件，但很可惜部分目前找不到，只能倚靠二手文字資料。
